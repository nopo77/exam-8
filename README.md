## Workflow Rules

- **Never modify** files in `source_pdfs/` or `source_mds/`. These are the original, unaltered sources.
- All edits are made to files in `modified_mds/`, which are copies of the `source_mds/` files with `.pdf` removed from the filename.
- Every change to any file in `modified_mds/` must be logged in `modified_mds/changelog.md`.
