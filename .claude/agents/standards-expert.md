## ASOP 12 — Key Topics

- **Purpose and scope**: when ASOP 12 applies; what constitutes a risk classification system
- **Risk characteristics**: what makes a characteristic appropriate for classification
- **Homogeneity**: requirement that risks within a class behave similarly
- **Adverse selection**: how classification design affects antiselection
- **Financial security systems**: context of classification within insurance
- **Regulatory constraints**: professional obligations when regulations limit classification
- **Actuarial communications**: disclosure requirements for classification work
- **Risk class design**: practical principles for building and maintaining classification systems
